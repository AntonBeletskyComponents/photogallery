ПРОЕКТ: PHOTOGALLERY.JS
ВЕРСИЯ: 1.0.0
АВТОР:  GEMINI
ЯЗЫК:   JAVASCRIPT (ES6+) / CSS3

-------------------------------------------------------------------------------
1. ОПИСАНИЕ ПРОЕКТА
-------------------------------------------------------------------------------
PhotoGallery.js представляет собой компактное решение для отображения графических
файлов на веб-страницах. Является функциональным аналогом Fotorama, не требующим
библиотеки jQuery или иных внешних зависимостей.

ОСНОВНЫЕ ВОЗМОЖНОСТИ:
* Автоматическая сборка галереи из списка тегов <img>.
* Поддержка жестов прокрутки (Swipe) и управления с клавиатуры.
* Адаптивное соотношение сторон через CSS-переменные.
* Навигация: миниатюры (thumbnails) или индикаторы (dots).
* Полноэкранный режим просмотра.

-------------------------------------------------------------------------------
2. ПОРЯДОК УСТАНОВКИ
-------------------------------------------------------------------------------
Для корректной работы системы необходимо подключить следующие файлы:
```
   <link href="photogallery.css" rel="stylesheet">
   <script src="photogallery.js"></script>
```
-------------------------------------------------------------------------------
3. ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ (DATA-ATTRIBUTES)
-------------------------------------------------------------------------------
Настройка производится через атрибуты контейнера .photo-gallery:

ПАРАМЕТР          ЗНАЧЕНИЕ ПО УМОЛЧАНИЮ     ОПИСАНИЕ
-------------------------------------------------------------------------------
data-nav          "dots"                    Тип навигации: dots/thumbs/false
data-ratio        "16/9"                    Соотношение сторон (Ш/В)
data-loop         false                     Зацикливание списка
data-autoplay     false                     Автопрокрутка (мс)
data-arrows       true                      Отображение стрелок
data-thumbwidth   80                        Ширина превью (пикс)
data-thumbheight  60                        Высота превью (пикс)
data-click        true                      Переход по клику на фото

-------------------------------------------------------------------------------
4. ПРИМЕР РЕАЛИЗАЦИИ
-------------------------------------------------------------------------------
Ниже приведен стандартный блок разметки:
```
<div class="photo-gallery" data-nav="thumbs" data-ratio="16/10" data-loop="true">
    <img src="photo1.jpg" alt="Описание 1">
    <img src="photo2.jpg" alt="Описание 2">
    <img src="photo3.jpg" alt="Описание 3">
</div>
```
-------------------------------------------------------------------------------
5. ПРИНЦИП РАБОТЫ СКРИПТА
-------------------------------------------------------------------------------
1. ИНИЦИАЛИЗАЦИЯ: Поиск элементов с классом .photo-gallery.
2. ТРАНСФОРМАЦИЯ: Чтение <img>, создание DOM-структуры Stage и Track.
3. СТИЛИЗАЦИЯ: Установка CSS-переменных для сохранения пропорций.
4. СОБЫТИЯ: Регистрация слушателей Touch, Mouse и Keyboard.
